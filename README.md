# BookAnInstructor MVP

A starter marketplace web app for Dance Instructors in Brisbane.

## Stack
- Next.js
- Supabase
- Vercel

## Setup
1. Create a Supabase project
2. Run `supabase/schema.sql` in Supabase SQL Editor
3. Copy `.env.example` to `.env.local`
4. Add your Supabase URL and anon key
5. Run `npm install`
6. Run `npm run dev`

## Pages
- `/` landing page
- `/instructors` browse dance instructors
- `/post-job` client request form
- `/instructor-signup` instructor application form
- `/admin` simple admin view placeholder
