# BookAnInstructor Email Templates

Edit these files directly in VS Code.

Each template uses simple variables wrapped in double braces, for example:

`{{client_first_name}}`

These are draft operational templates. Marketing, legal, dispute, refund, and contract wording should be reviewed before being automated.
