# BookAnInstructor Category Landing Pages Plan

## Goal

Create focused landing pages for major instructor categories, each with its own SEO keywords, page copy, popular styles, and calls to action.

The main site `bookaninstructor.com` should act as the marketing layer. The app `app.bookaninstructor.com` remains the booking platform.

Primary calls to action:

- Find an instructor
- Post a job
- Become an instructor

## Core Landing Page Pattern

Each category page should follow the same proven structure:

1. Hero section tailored to the category.
2. Clear value proposition.
3. Three main buttons:
   - Post a Job
   - Find an Instructor
   - Become an Instructor
4. How it works.
5. Who it is for.
6. Popular instructor types.
7. Why use BookAnInstructor.
8. Reviews / trust section.
9. Final call to action.

Each page should link into the app with the category prefilled where possible:

- `/instructors?category=Arts`
- `/post-job?category=Arts`
- `/instructor-signup?category=Arts`

The app may need category query support added later if it does not already filter by category.

## SEO Rules For Category Pages

Each category page should have:

- One clear H1 using the category.
- Natural body copy containing the main category terms.
- Internal links to app search and post-job pages.
- Internal links between related category pages.
- Image alt text that names the category.
- A unique meta title and description.
- JSON-LD for `WebPage` and `Organization`.

Avoid creating hundreds of thin pages too early. Start with six strong category pages, then add style-specific pages only when there is enough instructor supply and search demand.

## Google Ads Rollout

Do not send all categories into one campaign.

Recommended order:

1. Client bookings: Arts / dance focused.
2. Instructor signup: Arts / dance instructors.
3. Client bookings: business or health, depending on supply.
4. Client bookings: music, sports, tutors after inventory is ready.

Each category should have its own campaign or tightly themed campaign group once there is enough budget and data.

## Notes

The current homepage is strongest for dance and performing arts. It should either remain the main homepage during launch or become the `/arts` page later.

The phrase "music page with all health & music styles" appears to be a typo. Treat `/music` as music lessons only, while health and wellbeing belongs under `/health`.
