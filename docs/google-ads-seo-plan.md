## Google Ads Campaign Structure

Start with one Search campaign only.

Campaign name:

`AU | Search | Instructor Bookings | Leads`

Location:

Start with Brisbane and nearby Queensland areas while the marketplace is being tested. Expand later after instructor supply is strong.

Network:

Google Search only to start. Leave Display off for the first phase.

Goal:

Leads.

Primary conversions:

- Client clicks `Post a Job`
- Client submits a job request

Secondary conversions:

- Clicks `Find an Instructor`
- Clicks `Become an Instructor`
- Instructor profile submission

## Conversion Tracking

Before increasing spend, track these actions:

Primary:

- `post_job_click` on the marketing homepage
- `client_job_submitted` in the app

Secondary:

- `find_instructor_click`
- `become_instructor_click`
- `instructor_profile_submitted`

Future high-value conversions:

- `agreement_sent`
- `payment_received`
- `booking_completed`

## Tracking Notes

Google Ads requires a final URL that sends users to the correct landing page, and the landing page should load without errors for Google AdsBot. Conversion tracking should be set up before campaigns scale so Google can learn which keywords drive valuable actions.

## First 14-Day Test

Budget:

Start conservatively until conversion tracking proves clean. Suggested first test: AUD $20-$50 per day.

Review daily:

- Search terms
- Cost per click
- Conversion rate
- Which button users click
- Job quality
- Instructor supply match

Do not optimise for cheap clicks. Optimise for real posted jobs and paid bookings.
