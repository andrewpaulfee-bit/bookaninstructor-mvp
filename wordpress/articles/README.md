# BookAnInstructor WordPress Article Drafts

These files are paste-ready drafts for WordPress posts/pages on `https://bookaninstructor.com`.

## How To Publish Each Article

1. In WordPress, go to `Posts > Add New`.
2. Use the `SEO title` from the comment at the top as the post title or SEO title.
3. Use the `Slug` exactly as shown.
4. Paste the article HTML into a `Custom HTML` block, or paste the text into the normal editor and keep the headings.
5. Add the `Meta description` in the SEO plugin if available.
6. Publish or save as draft.
